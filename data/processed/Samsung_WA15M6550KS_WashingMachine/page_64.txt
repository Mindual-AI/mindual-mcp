## 고장신고 전 확인하기

### 점검문자 발생 시 대처하기

| 코드 | 진단 | 해결방법 |
|---|---|---|
| SE | 물이 빠지지 않을 경우 신호음과 함께 표시됩니다. |  - 배수호스 안에 찌꺼기가 끼어 있지 않은지 확인하세요.  - [동작/일시정지] 버튼을 누르면 다시 동작합니다. |
| UE | 탈수가 안되는 경우입니다. |  - 세탁물이 한쪽으로 치우친 경우 골고루 편 후, 뚜껑을 닫고 [동작/일시정지] 버튼을 누르면 재 동작 합니다.  - 세탁기를 수평으로 놓았는지 확인하세요. |
| ЧЕ | 급수 변화가 없는 경우 신호음과 함께 표시됩니다. / 급수가 완료되지 않았을 때 신호음과 함께 표시됩니다. |  - 수도꼭지를 열었는지, 단수가 아닌지, 얼지는 않았는지 확인하세요.  - 수도 급수량이 너무 적지 않은지 확인하세요.  - 급수호스 연결구 거름망에 찌꺼기가 끼어 있는지 확인하세요.  - [동작/일시정지] 버튼을 누르면 다시 동작합니다. |
| dE | 뚜껑이 열려있는 경우입니다. |  - 뚜껑이 열려 있지 않나 확인하세요.  - 뚜껑을 닫고 [동작/일시정지] 버튼을 누르면 동작 됩니다.  - 어린이 보호기능이 설정된 상태에서 뚜껑을 닫으면 'CL' 이 표시됩니다. 전원을 껐다가 켜면 다시 동작이 가능합니다. |
| CL (dE) | 어린이 보호기능이 동작된 경우입니다. |  - 어린이가 세탁조에 빠져 익사 할 우려를 방지하기 위한 기능입니다.  - 어린이 보호기능이 설정된 경우 뚜껑을 열고 30 초가 경과하면 'CL' 이 발생되며 동작이 되지 않습니다. (세탁조 내에 물이 있는 경우 배수가 됩니다.)  - 뚜껑을 닫고 전원을 껐다가 켜면 다시 사용할 수 있습니다.  - 어린이 보호기능 해제를 하고자 할경우 'CL' 표시가 아닌 상태에서 [헹굼횟수] + [탈수시간] 버튼을 동시에 3초 이상 누르세요. |
| 8E/P8/3E 3E2/3E/4E 8E2/PE/E LE |  |  - 고장일 수 있으니 가까운 삼성전자 대리점이나 서비스센터에 문의하세요. |
